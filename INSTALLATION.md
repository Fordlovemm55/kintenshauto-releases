## 🚀 วิธีติดตั้ง (3 ขั้นตอน)

### ขั้นตอนที่ 1: ดาวน์โหลดโปรแกรม

1. เปิดลิงก์ที่ได้รับมาจากผู้พัฒนา
2. ดาวน์โหลดไฟล์ตามระบบของคุณ:
   - **Windows**: `KINTENSHAUTO-Setup-1.0.0.exe`
   - **Mac**: `KINTENSHAUTO-1.0.0.dmg`
   - **Linux**: `KINTENSHAUTO-1.0.0.AppImage`

### ขั้นตอนที่ 2: ติดตั้ง

#### Windows
1. ดับเบิลคลิกไฟล์ `.exe`
2. ถ้า Windows SmartScreen เตือน "Windows protected your PC":
   - กด **"More info"**
   - แล้วกด **"Run anyway"**
   (เกิดจากโปรแกรมยังไม่ได้ code-sign — ปลอดภัยถ้าดาวน์โหลดจากแหล่งที่เชื่อถือได้)
3. ทำตาม wizard — เลือกโฟลเดอร์ติดตั้ง → กด Install
4. รอ 30 วินาที ถึง 2 นาที

#### Mac
1. เปิดไฟล์ `.dmg`
2. ลาก ไอคอน KINTENSHAUTO ไปที่โฟลเดอร์ Applications
3. ถ้าเปิดครั้งแรกและ Mac เตือน "cannot be opened because it is from an unidentified developer":
   - เปิด **System Preferences** → **Security & Privacy**
   - กด **"Open Anyway"** ที่ด้านล่าง

#### Linux
```bash
chmod +x KINTENSHAUTO-1.0.0.AppImage
./KINTENSHAUTO-1.0.0.AppImage
```

### ขั้นตอนที่ 3: ติดตั้ง Google Chrome (ถ้ายังไม่มี)

ระบบต้องใช้ Chrome เพื่อ automate Facebook

1. ไปที่ https://www.google.com/chrome/
2. ดาวน์โหลดและติดตั้ง Chrome แบบปกติ
3. **ไม่ต้อง login** ใน Chrome — โปรแกรมจะใช้โปรไฟล์แยก

## 🎮 เริ่มใช้งานครั้งแรก

### 1. เปิดโปรแกรม

ดับเบิลคลิกไอคอน KINTENSHAUTO บน Desktop

รอสักครู่ คุณจะเห็นหน้าจอ splash พร้อมตัวอักษร **剣天照** ก่อนเข้าโปรแกรม

### 2. Setup Wizard (การตั้งค่าครั้งแรก)

โปรแกรมจะเข้า **Setup Wizard** อัตโนมัติ มี 5 ขั้นตอน:

#### ขั้นที่ 1 — ยินดีต้อนรับ 始め
- อ่านคำเตือนสำคัญ
- กด "เข้าใจแล้ว เริ่มเลย"

#### ขั้นที่ 2 — ตรวจสอบเครื่องมือ 道具
- โปรแกรมจะเช็คว่ามีเครื่องมือครบไหม
- **FFmpeg, yt-dlp, fpcalc** — กด "ติดตั้งอัตโนมัติ" โปรแกรมจะดาวน์โหลดให้เอง (ประมาณ 150 MB, ใช้เวลา 2-5 นาทีตามความเร็วเน็ต)
- **Chrome** — ต้องติดตั้งเอง (ดูขั้นตอนที่ 3 ด้านบน)

> 💡 ถ้า "ติดตั้งอัตโนมัติ" ล้มเหลว ดูหัวข้อ "การแก้ปัญหา" ด้านล่าง

#### ขั้นที่ 3 — เพิ่มเฟส Facebook 盟友 (ข้ามได้)
- ใส่ชื่อกำกับ (เช่น "เฟสหลัก 1")
- Facebook username/email + รหัสผ่าน
- Proxy (แนะนำ — ช่วยลดความเสี่ยงโดนแบน)
- **ข้อมูลจะถูก encrypt เก็บในเครื่องคุณ ไม่ส่งออกไปไหน**

#### ขั้นที่ 4 — ตั้งค่า AI 知恵 (ข้ามได้)
- เลือก provider: **OpenAI / Claude / Gemini**
- ใส่ API key (ดูวิธีหาด้านล่าง)
- ถ้าไม่ตั้ง AI ระบบจะใช้ template แคปชั่นแทน (ก็ใช้ได้ปกติ)

#### ขั้นที่ 5 — พร้อมใช้งาน 出陣
- กด "เข้าสู่โปรแกรม"

### 3. หน้าหลัก

คุณจะเห็น **Dashboard** พร้อมพื้นหลังซามูไรเคลื่อนไหว ✨

## 🔑 วิธีหา API Key

### OpenAI (GPT-4o, GPT-4o-mini)
1. ไปที่ https://platform.openai.com/api-keys
2. Login / สมัครใหม่ (ต้องเติมเงินขั้นต่ำ $5)
3. กด "Create new secret key"
4. คัดลอก key (ขึ้นต้นด้วย `sk-...`)

**ราคา**: GPT-4o-mini ประมาณ $0.15 ต่อ 1M input tokens (ถูกมาก — คลิปละไม่ถึง 10 สตางค์)

### Claude (Anthropic)
1. ไปที่ https://console.anthropic.com/settings/keys
2. Login / สมัครใหม่
3. กด "Create Key"
4. คัดลอก key (ขึ้นต้นด้วย `sk-ant-...`)

### Google Gemini
1. ไปที่ https://aistudio.google.com/apikey
2. Login ด้วย Google account
3. กด "Create API key"
4. คัดลอก key

**Gemini มี free tier** — ใช้ได้วันละหลายพันครั้งฟรี!

## 📚 คำแนะนำการใช้งานหลังเข้า app

### ลำดับการตั้งค่า (แนะนำ)

1. **จัดการเฟส** → เพิ่มเฟส FB (ถ้ายังไม่ได้เพิ่มใน wizard)
2. **แบนเนอร์** → อัปโหลดรูปแบนเนอร์ + ตั้งตำแหน่ง
3. **คอมเม้นอัตโนมัติ** → สร้าง template คอมเม้น
4. **AI แคปชั่น** → ตั้ง prompt ต่อเพจ
5. **หน้าหลัก** → เลือกเพจ + keyword → กด **RUN** ⚔

### ครั้งแรกที่ login FB

- โปรแกรมจะเปิด Chrome ขึ้นมา
- **คุณต้อง login เอง** (โปรแกรมไม่พิมพ์รหัสให้ — เพื่อความปลอดภัย)
- ถ้า FB เด้ง 2FA, checkpoint, captcha → **ยืนยันเอง**
- เมื่อเข้า FB หน้าหลักได้แล้ว โปรแกรมจะบันทึก session อัตโนมัติ
- ครั้งต่อไปจะไม่ต้อง login อีก

## ⚠ การแก้ปัญหาที่พบบ่อย

### "ติดตั้ง dependencies ล้มเหลว"
**สาเหตุ**: เน็ตขาด หรือ firewall บล็อก GitHub

**วิธีแก้**:
1. ปิด firewall ชั่วคราว ลองใหม่
2. ถ้ายังไม่ได้ — ดาวน์โหลดแบบ manual:
   - FFmpeg: https://www.gyan.dev/ffmpeg/builds/ — ดาวน์โหลด `ffmpeg-release-essentials.zip`
   - yt-dlp: https://github.com/yt-dlp/yt-dlp/releases/latest
   - วางไว้ที่ `C:\Users\YourName\AppData\Roaming\KINTENSHAUTO\bin\` (สร้างโฟลเดอร์เอง)
3. เปิดโปรแกรมใหม่ → กด "ตรวจซ้ำ"

### "Chrome/Chromium ไม่เจอ"
**วิธีแก้**: ติดตั้ง Google Chrome จาก https://www.google.com/chrome/

### "โปรแกรมเปิดไม่ขึ้น" (Windows)
**วิธีแก้**:
1. เปิด Task Manager → จบ process `KINTENSHAUTO.exe` ทั้งหมด
2. เปิดโปรแกรมใหม่

### "FB ให้ยืนยันตัวตน ทุกครั้งที่เปิด"
**สาเหตุ**: IP ไม่สม่ำเสมอ หรือ browser fingerprint เปลี่ยน

**วิธีแก้**:
- ใช้ Proxy ที่อยู่จังหวัดเดียวกัน
- ใช้ IP เดียวกันนาน ๆ (ไม่เปิด VPN สลับไปมา)

### "คลิปติดลิขสิทธิ์ทุกครั้ง"
**วิธีแก้**:
- ใช้ **Set 2** (โปรแกรมสร้างอัตโนมัติ) — ปรับความเร็ว, mirror, pitch
- หลีกเลี่ยงคลิปที่มีเพลงดังจาก label ใหญ่

### "โปรแกรม crash"
**วิธีแก้**:
1. เปิด File Explorer → พิมพ์ `%APPDATA%\KINTENSHAUTO\logs\`
2. ส่งไฟล์ `backend.log` และ `app.log` ให้ผู้พัฒนา

### "ข้อมูลของฉันเก็บอยู่ที่ไหน?"
- **Windows**: `C:\Users\YourName\AppData\Roaming\KINTENSHAUTO\`
- **Mac**: `~/Library/Application Support/KINTENSHAUTO/`
- **Linux**: `~/.config/KINTENSHAUTO/`

มีไฟล์:
- `kintenshauto.db` — database ทุกอย่าง (ตั้งค่า, คลิป, logs)
- `chrome-profiles/` — Chrome profile แยกต่อเฟส
- `overlays/` — รูปแบนเนอร์
- `downloads/` — คลิปที่ดาวน์โหลด
- `logs/` — log ไฟล์

## 🔐 ความปลอดภัยและความเป็นส่วนตัว

- **API keys ถูก encrypt** ด้วย AES-256 ก่อนเก็บใน DB
- **รหัสผ่าน FB ถูก encrypt** เช่นกัน
- **ไม่มีข้อมูลใดส่งออกนอกเครื่องคุณ** (ยกเว้น request ไปหา AI provider ที่คุณตั้งเอง และ FB ตามการใช้งานปกติ)
- **ไม่มี telemetry / analytics**

> ⚠ เพื่อความปลอดภัยสูงสุด: ใช้รหัสผ่าน Windows ล็อคเครื่องด้วย
